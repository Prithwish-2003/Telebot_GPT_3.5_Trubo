# Telebot_GPT_3.5_Trubo

## Prereqisites
```bash
1. python 3.7 env
2. Telegram Account
3. OpenAI Account For API Key
```
## How To Run -->
```bash
conda create -n env python=3.7 -y
```

```bash
conda activate env
```

```bash
pip install -r requirements.txt
```

```bash
python app.py
```
